Summarize

การแสดงผลเว็บไซต์ของเบราว์เซอร์

  1. Parsing การสร้าง object จาก code ที่ได้รับจาก server ประกอบด้วย 2 ขั้นตอนย่อยคือ
    • Tokenization แยก code ออกเป็นส่วนๆตามประเภท
      Start tag: tag open, tag name, tag close
      End tag: close tag open, tage name, tag close
    • Parsing นำ token แต่ละส่วนนำมาตรวจสอบกับ HTML syntax
      Reentrant คือ parsing process ที่สามารถเกิด interrupted ได้
  2. Render tree
    ถูกสร้างขึ้นมาจาก DOM และ CSSOM โดยรวม DOM เข้ากับ CSSOM
  3. Layout
    อาจเรียกว่า Reflow คือการหาตำแหน่ง ขนาดของnode(หรือ element) ใน render tree ขนาดของ node ประกอบด้วย width กับ height
  4. Paint
    เป็นขั้นตอนที่ต้องใช้ข้อมูลทั้งหมดที่ได้จากการrender tree จะรวม render tree เป็น layers มีการสร้าง bitmap จากแต่ละlayer และแสดงผลบนหน้าจอ
    bitmap คือ การอัพโหลดไปยัง GPU เพื่อสร้าง texture

DOM (Document Object Model) ตัวแสดงคุณสมบัติและความสัมพันธ์ระหว่างแต่ละ object ในหน้าเว็บ(document) ซึ่งก็จะประกอบไปด้วย properties กับ methods ของแต่ละ object พร้อมกับความสัมพันธ์กับ object อื่นๆ

CSSOM (CSS Object Model) คล้าย DOM แต่เป็นการสร้าง object model จาก CSS ของเว็บ ที่มีทั้ง properties และความสัมพันธ์กับ object อื่นๆ (Cascading ของ CSS)
